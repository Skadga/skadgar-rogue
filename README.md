## 📖 Описание
Skadgar — это изометрическая RPG в стиле roguelike. Исследуйте парящий остров, сражайтесь с монстрами, собирайте лут и проходите данжи.

### Особенности:
- ⚔️ Динамические бои в реальном времени
- 🎒 Система инвентаря и экипировки
- 🌀 Портал в опасное подземелье
- 🛡️ Прокачка характеристик
- 🤖 Авто-атака для удобного фарма
- 💾 Сохранение прогресса в 3 слота

## 🎯 Управление
| Клавиша | Действие |
|---------|----------|
| ЛКМ (зажать + движение) | Перемещение |
| ПКМ | Атака / вход в портал |
| `I` | Инвентарь |
| `G` | Магазин (у торговца) |
| `E` | Вход в портал |
| `R` | Перезапуск игры |
| `ESC` | Пауза / закрыть меню |

## 🧙‍♂️ Торговец
Найдите Лорда Альдрика рядом с порталом. Он может:
- 🩸 Продать зелья здоровья
- 🛡️ Улучшить броню
- ⚔️ Увеличить урон
